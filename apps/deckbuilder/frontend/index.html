    Import deck list

    Paste one card per line. Quantity, set code in (parens), and collector number are ignored — only the card name is used. Anything that can't be added cleanly (wrong type, off-colour, over quota) still imports and is flagged in the list.

    Import custom cards

    Paste JSON — an array of cards (or {"cards": [...]}), each with name, type (instant / sorcery / enchantment), mana_cost, effect, and optional flavour (an in-character description of how the effect works — the card editor's Flavour field). Cards are added to the current deck; nothing is replaced. Effect text is run through the same translation pass as MTG imports — anything it can't parse is flagged for hand-authoring. See CUSTOM_CARD_SCHEMA.md for the full format.
